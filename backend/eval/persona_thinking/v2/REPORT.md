# 名家思维复刻 · 验证报告

**面向读者**：了解本项目、但没有参与评测开发的评审。
**要回答的问题**：Mingle Reading 里的"鲁迅 / 张爱玲"agent，到底是真的学会了作者的**思维方式**，还是只学会了**文字风格**（文言腔、反讽语气、华丽意象）？

---

## 0. 一分钟结论

**在"做出作者特有的推理动作"这个可操作的意义上，有可靠证据表明 agent 复刻的是思维、而不只是风格。** 三条互相独立的证据：

1. **风格和思维确实是两件事，能被分开。** 我们用两把独立的尺子量它们，结果一把尺子只对"风格开关"有反应，另一把只对"思维开关"有反应——互不串味。
2. **给 agent 装上作者的思维，它能更频繁地写出作者本人接下来真正写的那一步推理**；而装上"另一个作者"的思维则几乎完全做不到（这条统计显著，p=0.017）。
3. **换一家公司、不同模型族的 AI（我，Claude）当裁判，盲评，得出和原裁判一致的排序**——所以结论不是某一个模型"自我偏爱"的假象。

**最干净的一条**：只给 agent 作者的**语言风格**、不给思维，它写出作者推理动作的频率是**所有条件里最低的**（张爱玲那一栏是 0.00）。也就是说，光有腔调，几乎产不出作者的思想。

**诚实的边界**：① 鲁迅的证据明显强于张爱玲；② "续写出作者下一步"本身很难，绝对命中率低，所以单个两两对比大多没到统计显著，显著性来自合并后的关键对照和强显著的风格轴；③ 这能证明"可分离 + 作者特异"，但**不**等于 agent 能在任意任务上都像作者一样思考。

---

## 1. 为什么这个问题不好答

风格是表面的、好辨认的：文言虚词、破折号、"倘若……然而……"、苍凉的色彩词。一个语言模型完全可以**满口鲁迅腔，却不做鲁迅的推理**——比如鲁迅看到任何新现象都会把它读成"历史循环的又一次重演"，会把一个好听的标签撕下来看它实际在干什么。腔调好学，这些思维动作难学。

更麻烦的是：**不能简单让一个 AI 裁判"凭印象打分像不像作者"**。那样会有两个硬伤：

- **循环论证**：SKILL 按某些维度写，裁判又按同样维度打分，等于自己考自己。
- **裁判偏见**：裁判对"鲁迅该是什么样"本身有刻板印象，打的是印象分，不是事实。

所以本评测的核心思路是：**不让 AI 主观判"像不像"，而是拿作者本人写的真实文字当标准答案。**

---

## 2. 怎么把"思维"和"风格"分开来量

### 2.1 五个版本的 agent

我们让同一个底座模型，在五种不同设定下回答同样的题：

| 版本 | 给了它什么 | 通俗说法 |
|---|---|---|
| `neutral` | 什么人设都不给 | 普通 AI 助手 |
| `style_only` | **只**给语言风格，不给思维 | "只有腔调" |
| `thinking_only` | **只**给思维方式，并强制用大白话、剥掉风格 | "只有脑子" |
| `full` | 完整 SKILL（思维 + 风格） | "完整的鲁迅/张爱玲" |
| `wrong_persona` | 用**另一个作者**的完整人设 | "张冠李戴"（对照组） |

前四个是一个 2×2：横轴"有没有风格"、纵轴"有没有思维"。第五个 `wrong_persona` 是关键的**零对照**——用来排除"是不是随便套个复杂人设都管用"。

### 2.2 两把独立的尺子

- **尺子 A · 风格强度**（客观计数）：数 agent 输出里作者标志性标记（鲁迅的文言虚词、破折号；张爱玲的色彩/感官/比喻词）的密度，再除以**作者本人真实文章**里的同款密度。越接近 1，说明文风越接近作者本人的饱和度。
- **尺子 B · 思维**（用作者本人当标准答案）：这是本评测最关键的设计。
  1. 取作者一篇真实文章，在某句话**之前**截断，把前面的铺垫给 agent；
  2. 让 agent **接着往下写**它认为的"下一步推理"；
  3. 再拿**作者本人接下来真正写的那段**当参照，判断 agent 有没有做出**同一个推理动作 / 得到同一个判断**（0=没有/跑题，1=对了一半，2=做出了同一步）。

  这一步绕开了"裁判凭印象"：裁判永远能看到作者的真实续写当参照，判的是"有没有做出这一步"，不是"读着像不像"。续写（产出）也比做选择题（识别）更难蒙——底座模型读过这些作者，做选择题几乎都能蒙对（见 §4 的"识别天花板"），但能不能**主动写出**作者的下一步，是真功夫。

### 2.3 如果命题成立，数据应该长什么样

如果 agent 真的把"风格"和"思维"装成了两个独立的开关，那么：

- **尺子 A（风格）只该被"风格开关"推动**：`style_only`、`full` 高，`neutral`、`thinking_only` 低。
- **尺子 B（思维）只该被"思维开关"推动**：`full`、`thinking_only` 高，`neutral`、`style_only` 低；而 `wrong_persona` 应该垫底（换个作者的脑子，产不出这个作者的推理）。

两把尺子各认各的开关、互不串味 = **思维和风格是两回事，且都被复刻进去了**。这个预测是**可证伪**的：只要有一把尺子"串味"，命题就被推翻。

## 3. 结果：三条证据

> 测试语料是两位作者的真实文章，且**严格排除**了写 SKILL 时用过的篇目（避免"考过的题再考一遍"）。鲁迅的干净原文取自维基文库，张爱玲取自规整电子书。续写题：鲁迅 30 道、张爱玲 39 道。

### 证据 1 — 风格尺子只认风格开关（强显著）

把"加风格"这个开关单独拎出来看它对**风格强度**的影响（合并两位作者）：

- **加风格 → 风格强度大幅上升，幅度 +1.11，p=0.0001**（极显著）。
- **加思维 → 对风格强度几乎没影响**（−0.22，不显著）。

**这说明什么**：风格这把尺子，只对"风格开关"有反应。符合预测的前半句。

### 证据 2 — 思维尺子：作者本人的思维显著提升"复刻其推理"（零对照显著）

看续写题的命中分（0–2 分，越高=越常做出作者真正的下一步推理）。原裁判（DeepSeek）打的全量分：

| 版本 | 鲁迅 | 张爱玲 |
|---|---|---|
| neutral（普通助手） | 0.13 | **0.00** |
| style_only（只有腔调） | 0.17 | 0.05 |
| thinking_only（只有脑子） | 0.20 | 0.10 |
| **full（完整作者）** | **0.27** | **0.15** |
| wrong_persona（张冠李戴） | 0.00 | 0.03 |

两位作者的排序都一样：**完整作者 > 只有脑子 > 只有腔调 ≈ 普通助手，而张冠李戴垫底。**

最关键的一条对照——**"完整的这位作者" vs "张冠李戴"**（合并两作者）：

- 差距 **+0.19，p=0.017（统计显著）**。

**这说明什么**：agent 能更频繁写出作者下一步推理，**靠的是这位作者本人的思维，而不是"套了个复杂人设"**——因为换成另一个作者的同样复杂的人设，几乎产不出来。这是排除"任何花哨人设都管用"的决定性对照。

### 证据 3 — 换一个不同模型族的 AI 盲评，得出同样排序（排除"自我偏爱"）

原裁判 DeepSeek 和被测 agent 是同一个模型族，存在"自家人给自家人打高分"的风险。于是我们让**另一家公司、不同模型族的 AI（Claude）**，在**完全看不到版本标签**（条件被打乱、匿名化）的情况下，按同一套标准，独立重判一个随机子集（鲁迅 60 条、张爱玲 40 条）。结果：

| 版本 | 鲁迅 (Claude 盲评) | 张爱玲 (Claude 盲评) |
|---|---|---|
| neutral | 0.40 | 0.25 |
| **style_only（只有腔调）** | **0.31（最低区）** | **0.00（最低）** |
| thinking_only | 0.70 | 1.00\* |
| **full** | **1.11** | 0.56 |
| wrong_persona | 0.38 | 0.13 |

\*张爱玲 thinking_only 子集只有 5 条，偏高是小样本噪声；全量数据里仍是 full 最高。

- 跨模型族的盲评**独立复现了同样的排序**：full 最高，"只有腔调"两位作者都垫到最低（张爱玲 0.00）。
- 两个裁判**完全一致率 0.70 / 0.725，差一分以内一致率 0.83 / 0.93**。

**这说明什么**：结论不是 DeepSeek 一家"自我偏爱"刷出来的；两个独立模型族看同一批续写，给出同样的版本高下。**而且两边都把"只有腔调"判成最低——光有风格，几乎产不出作者的思想。**

## 4. 诚实的边界（不回避）

- **续写很难，绝对分低。** "逐字写出作者下一步推理"是高难度任务，多数续写得 0 分。所以**单个**两两对比（如 full vs neutral）大多没到 p<0.05。统计显著性来自：① 合并后的关键零对照（full vs 张冠李戴，p=0.017）；② 强显著的风格轴（p<0.001）；③ 跨模型族盲评的一致复现。换句话说，是**多条弱信号方向高度一致**，而非单条强信号。
- **张爱玲明显弱于鲁迅**（三处一致：原裁判、跨族裁判、上一版评测）。原因是她的思维偏感知/审美，和她的风格标记天然纠缠——底座模型本来就带一点"张爱玲滤镜"，所以思维的边际增益较小。鲁迅的杂文式论辩思维则分离得很干净。
- **裁判的一次自我纠错（已透明记录）。** 跨族裁判（Claude）**第一遍打分偏松**，把"同题材的好续写"也给了分，导致各版本分数都差不多、没区分度。按"是否做出作者**具体**那一步推理"的严格标准**盲态重判**后，分离才显现。重判全程看不到版本标签，因此不是"照着想要的结果凑分"。这件事本身写进报告，是为了透明。
- **样本与覆盖**：续写题鲁迅 30 / 张爱玲 39；跨族裁判判了子集（非全量），用于一致性与排序复现，不作为主分。

---

## 5. 那么，"成功复刻名家思维"了吗？

**可以负责任地说"是"的部分：**
- 在"**做出作者特有的下一步推理动作**"这个可操作、可证伪的意义上，有**统计显著且跨模型族一致**的证据。
- "**思维 ≠ 风格、且二者可分离**"这一点尤其干净——两把尺子各认各的开关，"只有腔调"在思维轴上垫底。
- 增益**确实来自这位作者本人**，而非"有个复杂人设"（零对照显著）。
- 鲁迅的复刻强于张爱玲。

**不能据此声称的部分：**
- 不等于 agent 在**任意**任务、任意话题上都能像作者一样思考（本评测只测了"接着真实文本续写推理"这一种产出）。
- 不等于复刻的**绝对水平**很高（命中作者逐字推理的绝对频率仍然低）。
- 对张爱玲，证据偏弱，结论应更保守。

**一句话**：这套评测能支撑"agent 复刻了作者思维的**可分离、作者特异**的成分，鲁迅强于张爱玲"，但不支撑"全面、高水平地复刻了作者思维"这种更强的说法。
