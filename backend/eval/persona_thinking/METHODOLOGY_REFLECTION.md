# 方法论反思：这套评测能否严谨地说明"复刻了名家思维"？要不要加 bge-m3 语义相似度？

本文回答两个问题：
1. 当前评测体系，在多大程度上**严谨地**支撑"agent 复刻了名家思维"这一命题？
2. 是否需要再引入**文本余弦相似度（bge-m3 embedding）**为评测体系补充证据？

---

## 一、这套评测的严谨性：它证明了什么，没证明什么

### 1.1 它做对的、确实严谨的地方

- **不靠"裁判凭印象判像不像"，而用作者本人的真实文字当标准答案。** 这是相对 v1（rubric 主观打分）
  最关键的进步，直接消除了两个硬伤：循环论证（按维度写、按维度打分）和裁判刻板印象。
- **可证伪的析因设计。** 2×2（风格开关 × 思维开关）+ wrong_persona 零对照，预测了一个**具体的、可被推翻的**
  数据形态（两把尺子各认各的开关）。只要有一把尺子串味，命题就被否。它没有被否。
- **作者特异性有零对照背书。** "完整这位作者" vs "换另一作者人设"显著（p=0.017）——增益来自**这位**作者的
  思维，而不是"套了个复杂人设"。这是很多 persona 评测缺的对照。
- **跨模型族盲评。** 同族裁判会自我偏爱；换不同族的裁判盲评、复现同一排序，把结论从"一个模型说"升级为
  "两个独立模型族都说"。三裁判（加 Codex）会进一步夯实。
- **风格与思维可分离**这一结论尤其干净（风格主效应 p=0.0001，思维对风格几乎无效；"只有腔调"在思维轴垫底）。

### 1.2 它的真实边界（这些必须诚实写在结论里）

- **构念效度的核心假设值得追问**："能续写出作者下一步的推理动作" ≈ "复刻了作者的思维"——这是**一种**操作化，
  不是唯一定义。作者真实的下一句本身有偶然性（他本可以那样写、也可以这样写）；"是否同一步"仍是判断，
  只是被 reference 锚定了而已。所以这测的是**"复刻作者特有推理动作"这一可操作侧面**，不是"思维"的全部。
- **外部效度有限**：续写题取自**散文/杂文**，而产品实际场景是"陪读时就用户选段发表评论"。在续写上复刻得好，
  不直接等于在真实陪读对话里复刻得好。两者相关但不等同。
- **功效弱、绝对分低**：逐字命中作者下一步本就难，多数续写得 0；单格两两对比大多不显著，显著性靠**合并**后的
  零对照与强显著的风格轴撑着。这是"多条方向一致的弱信号"，不是"一条铁证"。
- **裁判主观性未被完全消除**：reference-based 已大幅降低主观，但"是否做出同一步"仍是判断——证据是跨族
  **一致率 ~0.70（完全）/ ~0.85–0.93（±1）**，不是 1.0。
- **样本规模**：续写题鲁迅 30 / 张爱玲 39，作者各 ~16 篇 held-out。够看趋势，不够看小效应。
- **张爱玲明显弱**：其思维偏感知/审美、与风格纠缠，证据弱于鲁迅，结论须更保守。

### 1.3 严谨性的总评

**够得上"严谨的初步证据"，够不上"决定性证明"。** 它能负责任地支撑一个**有界的**命题——
"在'做出作者特有推理动作'这一可操作、可证伪、跨模型族一致的意义上，agent 复刻了作者思维中**可与风格分离、且为作者特异**的成分；鲁迅强于张爱玲"——
而**不**支撑"全面、高水平地复刻了名家思维"这种无限定的强主张。对答辩而言，**这种带边界的诚实表述本身就是它的说服力来源**，
而不是弱点。要继续加强，优先级最高的是：①扩续写题量提功效；②补齐第三裁判（Codex）；③对续写判分做人工抽检校准；
④补一个贴近真实陪读场景的小测试集（外部效度）。

## 二、要不要加 bge-m3 文本余弦相似度？

评测过程中考虑过：是否再加一条 bge-m3 文本余弦相似度的证据——把真实原文段落嵌成"作者语义质心"，
再算各条件去风格输出到质心的余弦相似度。问题是：把它当作**思维**证据，靠谱吗？**最终决定不加**，理由如下。

### 2.1 先说结论

**不建议把"输出→作者质心"的余弦相似度当作思维轴的指标——那基本上测的是风格/题材邻近度，会把本评测好不容易
分离出来的"风格"重新混回"思维"里。** embedding 至多有**两个有限的**正当用途（见 2.3），但即便如此也只能算旁证。

### 2.2 为什么"到作者质心的余弦"不是好的思维证据

- **embedding 把题材、内容、风格、语域全搅在一起。** 余弦高，主要说明"这段文字落在作者的语义/题材/文风邻域"，
  **不**说明"它做出了作者的推理动作"。
- **同题材、相反论点的两段文字，余弦可以很近。** 而本评测的全部意义恰恰是区分"同题材的好续写"与"做出作者那一步"
  ——这正是 embedding 最不擅长的（它是"意义袋子"，不是"推理步骤探测器"）。
- **质心由与测试题不同话题的原文段落构成**，余弦里掺了大量"泛文学腔/话题漂移"信号。
- 一句话：它会**重新引入**本评测设计上要剔除的风格/题材混杂，让干净的双重分离变浑。

### 2.3 embedding 仅有的两个正当位置（即便要加，也只能放这里）

1. **作为续写裁判的自动化旁证：`cosine(candidate, gold)`。**
   不嵌到"作者质心"，而是嵌**同一道题**里 agent 的续写 vs 作者真实续写。因为两者都续自**同一段 setup**，
   题材被大致钉住，残余的余弦差异更偏向"它有没有往作者去的方向走"。它**便宜、确定、可复现、无裁判主观**，
   可作为 LLM 续写判分的**自动化交叉验证**：若 cosine(candidate,gold) 的分条件排序与裁判分一致，
   则裁判结论多了一条不依赖 LLM 的支撑。**但**仍要标注为"旁证"——余弦近≠同一步，余弦远≠不同步。
2. **作为 style-stripping / thinking_only 的"操作检验"（manipulation check）。**
   用 embedding 验证：`thinking_only` 的强制白话**确实**把文本推离了作者的表面文风邻域（到质心余弦下降），
   而 `style_only` 确实拉近了。这是对"风格开关是否真的拨动了"的客观自查，属于方法可信度，不属于思维证据。

### 2.4 优先级

若目标是**加强"思维"这一主张**，把人力/算力投到 ①扩题量 ②第三裁判 ③人工抽检 ④外部效度小测试集，
回报远高于做 embedding。embedding（即便是 2.3-①的 `cosine(candidate,gold)`）的优点只是**便宜且正交**
（不依赖任何 LLM 裁判），但它终究是**旁证**，不能、也不该取代 reference-based 续写判分作为思维轴的主证据。
**简言之：当前不加；即便日后加，也只能摆在"自动化旁证 + 操作检验"的位置，绝不当思维轴主指标。**

> 最终决定：**本评测不引入 bge-m3 余弦相似度**。它既不能当思维轴的主指标（2.2），其唯一正当用法
> （2.3 的 `cosine(candidate, gold)` 自动化旁证）也只是"锦上添花"的弱证据，而当前的瓶颈在 §一 列出的
> 更高回报项（扩题量 / 第三裁判 / 人工抽检 / 外部效度集）。若日后重启，也只应放在"自动化旁证 + 操作检验"
> 的位置，绝不充当思维轴主指标。
