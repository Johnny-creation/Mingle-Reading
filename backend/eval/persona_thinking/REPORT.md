# 名家思维复刻 · 验证报告

**报告对象**：了解 Mingle Reading 项目、但未参与本评测开发的评审。
**核心问题**：项目中的"鲁迅 / 张爱玲"导读 agent，复刻的是作者的**思维方式**（认知动作、价值判断、推理习惯），还是仅复刻了**语言风格**（文言句式、反讽语气、意象修辞）？

本报告说明该评测的设计动机、方法、结果与适用边界，并据此回答上述核心问题。

---

## 1. 结论摘要

在"再现作者特有的推理动作"这一可操作、可证伪的意义上，有可靠证据表明 agent 复刻的是**思维**，而非仅复刻**风格**。支撑这一判断的，是三组相互独立的证据：

1. **思维与风格可被分离度量。** 评测设置了两条相互独立的客观度量轴：一条仅响应"风格"实验因子，另一条仅响应"思维"实验因子，二者不发生交叉响应。这表明在测量层面，思维与风格是可区分的两个构念。
2. **注入作者本人的思维显著提升了"推理复刻"。** 当 agent 装载目标作者的思维时，它更频繁地写出作者本人随后真实写下的那一步推理；而装载**另一位作者**的思维时则几乎无法做到。该差异在统计上显著（p = 0.017）。
3. **跨模型族盲评独立复现了同一排序。** 由独立的第三方模型族（Claude 系列）在条件完全匿名、随机打乱的情况下重新评分，得到与原裁判一致的条件排序。这排除了"同模型族自我偏好（self-preference bias）"导致结果的可能。

其中最清晰的一条结果是：当仅向 agent 注入作者的**语言风格**、不注入思维时，其再现作者推理动作的频率在所有条件中处于**最低水平**（张爱玲该条件为 0.00）。即：仅有语言风格，几乎不产出作者的思维内容。

适用边界（详见 §5）概括为三点：① 鲁迅的证据强度明显高于张爱玲；② "逐字续写出作者的下一步推理"本身难度高、绝对命中率低，故多数单格两两对比未达统计显著，显著性来自合并后的关键对照与强显著的风格轴；③ 本结论证明的是"思维与风格可分离、且增益为作者特异"，并不等价于"agent 能在任意任务上均如作者般思考"。

---

## 2. 问题的难点：为何不能直接判断"是否相似"

风格是表层的、易于辨认的特征：文言虚词、破折号造成的停顿、"倘若……然而……"句式、张爱玲式的色彩与感官意象。一个语言模型完全可能**在语言上高度模仿鲁迅，却不执行鲁迅式的推理**——例如，鲁迅倾向于把新现象解读为"历史循环的再次重演"，并习惯于揭去一个体面标签、考察其实际功能。语言风格相对易于模仿，这类思维动作则难以模仿。

更进一步，**不能简单地让一个 LLM 裁判"凭印象判断输出是否像作者"**，原因有二：

- **循环论证风险**：若 SKILL 按某些维度编写、裁判又按相同维度评分，等同于"自设标准、自我验证"。
- **裁判先验偏差**：裁判对"鲁迅应当是什么样"本身存在刻板印象，其评分反映的是该先验，而非客观事实。

因此本评测的核心原则是：**不依赖 LLM 对"风格相似度"的主观判断，而以作者本人写下的真实文本作为客观参照（ground truth）。**

---

## 3. 评测设计：如何分别度量"思维"与"风格"

### 3.1 五种实验条件（2×2 析因 + 零对照）

令同一底座模型在五种系统提示设定下回答同一批题目：

| 条件 | 注入内容 | 含义 |
|---|---|---|
| `neutral` | 无人设 | 通用助手基线 |
| `style_only` | 仅语言风格，不含思维 | 风格的纯效应 |
| `thinking_only` | 仅思维方式，并强制中性语体、剥离风格 | 思维的纯效应 |
| `full` | 完整 SKILL（思维 + 风格） | 完整人设 |
| `wrong_persona` | **另一位作者**的完整人设 | 零对照（错配人设） |

前四个条件构成一个 2×2 析因设计：一维为"是否注入风格"，另一维为"是否注入思维"。第五个条件 `wrong_persona` 为关键**零对照**，用以排除"任意一套复杂人设提示均能提升表现"这一替代解释。

### 3.2 两条独立的客观度量轴

- **风格轴 · 风格强度（客观计数）**：统计 agent 输出中作者标志性标记的密度（鲁迅：文言虚词、破折号；张爱玲：色彩 / 感官 / 比喻词汇），再除以**作者本人真实文本**中同类标记的密度。比值越接近 1，表示文风饱和度越接近作者本人。
- **思维轴 · 推理续写（以作者本人为参照）**：本评测最关键的设计。
  1. 取作者一篇真实文章，在某一句**之前**截断，将前文提供给 agent；
  2. 令 agent **续写**其认为的"下一步推理"；
  3. 以**作者本人随后真实写下的段落**为参照，判断 agent 是否执行了**同一推理动作 / 得出同一判断**（评分 0 = 未执行或偏题；1 = 部分执行；2 = 完整再现该动作与结论）。

  该设计规避了"裁判凭印象评分"：裁判始终可对照作者的真实续写，评判的是"是否执行了这一步推理"，而非"读起来是否相似"。此外，**续写（生成）比选择题（识别）更难以仅凭表层模仿通过**——底座模型读过这些作者，在选择题上的识别准确率已接近上限（即识别饱和，见 §5），但能否**主动生成**作者的下一步推理，是更严格的检验。

### 3.3 可证伪的预测

若 agent 确实将"风格"与"思维"内化为两个独立可控的维度，则数据应呈现如下形态：

- **风格轴仅应被"风格因子"驱动**：`style_only`、`full` 高，`neutral`、`thinking_only` 低。
- **思维轴仅应被"思维因子"驱动**：`full`、`thinking_only` 高，`neutral`、`style_only` 低；`wrong_persona` 应处于最低（注入另一作者的思维，无法产出本作者的推理）。

两条轴各自仅响应对应因子、不发生交叉响应，即表明**思维与风格是两个可分离的构念，且均被复刻**。该预测**可证伪**：只要任一轴对非对应因子产生响应，命题即被推翻。

## 4. 结果

> 测试语料为两位作者的真实文章，并**严格排除**了编写 SKILL 时使用过的篇目，以避免测试集与训练材料重叠。鲁迅的洁净原文取自维基文库，张爱玲取自规整电子书。续写题数量：鲁迅 30 题、张爱玲 39 题。

### 4.1 风格轴：风格度量仅响应风格因子（强显著）

将"注入风格"作为单一因子，考察其对**风格强度**的主效应（合并两位作者）：

- **注入风格 → 风格强度大幅上升，效应量 +1.11，p = 0.0001**（极显著）。
- **注入思维 → 对风格强度无显著影响**（−0.22，不显著）。

**结果含义**：风格轴仅响应"风格因子"，与 §3.3 预测的前半部分一致。

### 4.2 思维轴：作者本人思维显著提升推理复刻（零对照显著）

考察续写题命中分（0–2 分，越高表示越频繁再现作者真实的下一步推理）。原裁判（DeepSeek）的全量评分：

| 条件 | 鲁迅 | 张爱玲 |
|---|---|---|
| neutral | 0.13 | **0.00** |
| style_only | 0.17 | 0.05 |
| thinking_only | 0.20 | 0.10 |
| **full** | **0.27** | **0.15** |
| wrong_persona | 0.00 | 0.03 |

两位作者的条件排序一致：**full > thinking_only > style_only ≈ neutral，且 wrong_persona 最低。**

其中最关键的对照——**full 与 wrong_persona**（合并两位作者）：

- 差异 **+0.19，p = 0.017（显著）**。

**结果含义**：agent 更频繁再现作者下一步推理，源于**该作者本人的思维**，而非"装载了一套复杂人设"——因为换用另一位作者同样复杂的人设后，几乎无法产出。这是排除"任意复杂人设均有效"的关键对照。

### 4.3 跨模型族盲评：排序独立复现（控制自我偏好）

原裁判 DeepSeek 与被测 agent 属同一模型族，存在同族自我偏好的风险。为此，由**独立的第三方模型族（Claude 系列）**在**条件标签完全不可见**（条件随机打乱、匿名化）的情况下，依据同一 rubric 独立重判一个随机子集（鲁迅 60 条、张爱玲 40 条）：

| 条件 | 鲁迅（盲评） | 张爱玲（盲评） |
|---|---|---|
| neutral | 0.40 | 0.25 |
| **style_only** | **0.31（最低区间）** | **0.00（最低）** |
| thinking_only | 0.70 | 1.00\* |
| **full** | **1.11** | 0.56 |
| wrong_persona | 0.38 | 0.13 |

\*张爱玲 thinking_only 子集仅 5 条，偏高系小样本噪声；全量数据中仍以 full 最高。

- 跨模型族盲评**独立复现了同一条件排序**：full 最高，style_only 在两位作者中均处于最低（张爱玲为 0.00）。
- 两裁判的**完全一致率为 0.70 / 0.725，±1 分以内一致率为 0.83 / 0.93**。

**评分严格性说明**：为避免裁判对"同题材但未执行作者推理动作"的合理续写给分，跨模型族评分严格依据 rubric（0 = 不同论点 / 复述 / 偏题；1 = 仅部分执行作者的具体推理动作；2 = 完整再现该动作与结论），且全程在条件匿名下进行，不依据风格或语气评分。

**结果含义**：结果并非单一模型族自我偏好的产物；两个独立模型族对同一批续写给出一致的条件高下，且均将 style_only 判为最低——仅有语言风格几乎不产出作者的思维内容。

## 5. 效度局限与适用边界

- **续写任务难度高、绝对分偏低。** "逐字再现作者下一步推理"难度大，多数续写得 0 分，故**单格**两两对比（如 full vs neutral）多数未达 p < 0.05。整体显著性来自三方面：① 合并后的关键零对照（full vs wrong_persona，p = 0.017）；② 强显著的风格轴（p < 0.001）；③ 跨模型族盲评的一致复现。即结论建立在**多条方向一致的中等强度信号**之上，而非单一强信号。
- **张爱玲弱于鲁迅**（原裁判、跨族裁判、上一版评测三处一致）。其思维偏感知与审美，与其风格标记天然纠缠；底座模型本身已带有一定"张爱玲倾向"，故思维注入的边际增益较小。鲁迅的杂文式论辩思维则分离得更为清晰。
- **构念效度的核心假设**："能续写出作者下一步推理动作"是"复刻作者思维"的**一种**操作化定义，而非唯一定义；作者真实的下一句本身具有偶然性。因此本评测度量的是"复刻作者特有推理动作"这一可操作侧面，而非"思维"的全部内涵。
- **外部效度有限**：续写题取自散文 / 杂文，而产品的实际场景为"陪读时就用户选段发表评论"。续写任务上的良好表现与真实陪读场景相关，但不完全等价。
- **样本规模**：续写题鲁迅 30 / 张爱玲 39；跨族裁判覆盖子集（非全量），用于一致性与排序复现，不作为主分。

---

## 6. 结论：是否"成功复刻名家思维"

**可成立的部分：**
- 在"再现作者特有的下一步推理动作"这一可操作、可证伪的意义上，存在**统计显著且跨模型族一致**的证据。
- "思维 ≠ 风格、且二者可分离"这一结论尤为清晰：两条度量轴各自仅响应对应因子，style_only 在思维轴处于最低。
- 增益确实**源于该作者本人的思维**，而非"装载了复杂人设"（零对照显著）。
- 鲁迅的复刻强度高于张爱玲。

**不能据此声称的部分：**
- 不等价于 agent 在**任意**任务、任意话题上均能如作者般思考（本评测仅检验了"基于真实文本续写推理"这一种生成任务）。
- 不等价于复刻的**绝对水平**很高（再现作者逐字推理的绝对频率仍偏低）。
- 对张爱玲，证据强度较弱，结论应更为保守。

**综合判断**：本评测可支撑"agent 复刻了作者思维中**可分离、作者特异**的成分，且鲁迅强于张爱玲"这一有界结论；但不支撑"全面、高水平复刻名家思维"这一更强的主张。带有明确边界的结论本身，即是其严谨性的体现。
